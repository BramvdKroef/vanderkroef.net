genetic algorithm.

#+CAPTION: Three solutions the algorithm goes through to get to an optimal path.
[[file:images/map.svg]]

* Download
- [[https://github.com/BramvdKroef/Traveling-Salesman-Problem][Source on Github]]
  
* Usage

This runs the algorithm with an octagon:

#+BEGIN_EXAMPLE
python TravelingSalesman.py 26,2 62,2 87,26 87,62 62,87 26,87 2,62 2,26
#+END_EXAMPLE

You can also let it use random input:

#+BEGIN_EXAMPLE
python TravelingSalesman.py -r
#+END_EXAMPLE

Output a map to an image:

#+BEGIN_EXAMPLE
python TravelingSalesman.py -r -i map.svg
#+END_EXAMPLE
